System Settings


If you wish Compendium to use url's for node and background images and you are on a network that uses a proxy to connect to the Internet, you need to enter the proxy settings in the System Settings dialog. This will allow Compendium to access web images.

Tip: If you are not sure about this, just try using a url to an image file in a reference node image field and see if it draws OK. If it does not, then check your browser connection properties or network settings to find out if you are using a proxy.

This is not required if you are only using local image files for reference images and map images and backgrounds.

Figure 1. The Access Grid Meeting Setup dialog.

 

Field Purpose
Proxy Address (optional) If you are on a network that uses a proxy server for HTTP connections then enter the url for that here.
Proxy Port         (optional unless above is entered) The Port the HTTP proxy server uses.
