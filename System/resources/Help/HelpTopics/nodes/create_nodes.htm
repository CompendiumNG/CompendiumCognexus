Creating Nodes

There are three ways to create a node:

1. Via the node toolbar (see The Node Toolbar).

2. By selecting a node from the white-space right-click menu. On maps, the new node will be placed where your mouse pointer was when you opened the right-click menu. On Lists, the new node will appear at the bottom of the list. Below is an example of this menu


Figure 1. The white-space right-click menu, showing the node creation sub-menu


3. Via a keyboard shortcut, (see Node Types for a complete list of the keyboard shortcuts for creating new nodes)

For example, to create a Question node on a map, position your mouse pointer where you want the node to appear and type a ? or a I, or select Question Node from the right-click menu. To create say a Map node on a list, type an M, or select Map Node from the right-click menu, and the new node will appear at the bottom of the list.
