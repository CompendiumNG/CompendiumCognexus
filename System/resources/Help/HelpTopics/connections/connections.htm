Opening Connections

There are three connections that can be opened in Compendium: an IX Panel connection, a Jabber connection and a ClaiMaker connection.


Opening an IX Panel Connection

To open an IX Panel connection, go to the File Menu, select the 'Connections' option, and then the 'IX Panel Connection' option. You will then see a dialog window similar to the one in figure 1, below:


Figure 1. The IX Panel connections dialog window


There are four fields that you need to complete to open an IX Panel connection:


Field

Description

Server

Enter the address of the server you wish to use for this IX Panel Connection.

Username

Your user name on the IX Panel server you have entered above.

Password

Your password on the IX Panel server you have entered above.

Current Send Name

The name you wish IX Panel messages to be sent under.

Once you have entered all the correct information, press the 'Connect' button to open a connection. This will begin to open a connection and close the dialog window. Once the connection has successfully been opened, you will see a small message box pop up telling you of the fact, (see figure 2, below).


Figure 2. The IX Panel Connection confirmation message box


If at any stage you wish to close the connection, navigate to the IX Panel Connection dialog (as described above) and press the 'Disconnect' button, (which only becomes available when a connection is open).

If you want to close the dialog window without taking any action, press the 'Close' button.

For information on sending message, see IX Panels.

Opening a ClaiMaker Connection

To open a ClaiMaker connection, go to the File Menu, select the 'Connections' option, and then the 'ClaiMaker Connection' option. You will then see a dialog window similar to the one in figure 5, below:


Figure 5. The ClaiMaker connections dialog window


There is only one piece of information that you need to enter to establish a ClaiMaker connection, and that is the address of the server. Once you have entered this, press the 'Connect' button to open a connection. This will open a connection and then close the dialog window. No confirmation is required here as the connection is notional.

If at any stage you wish to close the connection, navigate to the ClaiMaker Connection dialog (as described above) and press the 'Disconnect' button, (which only becomes available when a connection is open).

For further information see ClaiMaker.
