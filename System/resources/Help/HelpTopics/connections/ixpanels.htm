IX Panels

I-X is an experimental technology developed at the University of Edinburgh to support teams in coordinating activities and producing artefacts like documents/reports. One component of the I-X suite of tools is the I-X Process Panel, which is like an active 'To Do' list. For more information see the I-X website for general information, and the I-P2 page for more specific information.

Compendium can send nodes to an I-X Process Panel, using the Jabber protocol. The following assumes that you have installed I-X Process panels and have an ID. Contact the Edinburgh team for details.

After Opening an IX Panel Connection, you will find that three menu options for sending messages to IX panels become available to you. The first is on The File Menu, see figure 1 below.
