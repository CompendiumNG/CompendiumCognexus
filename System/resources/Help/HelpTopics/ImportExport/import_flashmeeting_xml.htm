Import FlashMeeting XML

Below, in figure 1., is an example of the FlashMeeting XML Import dialog window. (see the Overview, for help on how to access this option).

N.B. The imported main meeting map will be placed inside the current view, so make sure the current view is the one you wish to import into.


Figure 1. The FlashMeeting XML Import dialog


There are ten import options which are all selected by default:


Option

Purpose

Import Keyword Data ?

This allows you to set whether you wish to import the keywords associated with the meeting.

If you do import them there will be a Keywords Map created containing a reference node pointing to the main FalshMeeting folksonomy and then a reference node for each of the keywords which will link to the folksonomy filetering listing meetings the keyword has been added to.

Import Play List Data ?

This allows you to set whether you wish to import the play list data associated with the meeting. This is a list of who spoke when.

If you do import this list there will be a 'Who spoke when' Map created containing a list of reference nodes listing at what time who spoke. When double-clicked each node will launch the Flashmeeting replay and start it at the time that person spoke.

Import URL Data ?

This allows you to set whether you wish to import URL data associated with the meeting. this will not only include URL specifically visited during the meeting, but it will also extract any URL's burried in Chat or annotation messages.

If you do import this list there will be a 'URLs' Map created containing a reference node for each url which will launch the browser to that url when clicked. Also, linked to each of these, there will be another reference node which when launch will start the meeting replay at the time at which the url was originally surfed, discussed or annotated.

Import Attendee Data?

This allows you to set whether you wish to import the list of Attendees associated with the meeting. This is a list of the people in the meeting based on the names they entered..

If you do import this list there will be an 'Attendees' Map created containing a list of note nodes listing the names of the attendees.

Import Chat Data?

This allows you to set whether you wish to import the chat messages associated with the meeting. This is a list of chat message writing during the meeting and who wrote them.

If you do import this list there will be a 'Chat' Map created containing a list of reference nodes listing at what time who wrote what chat message. When double-clicked each node will launch the Flashmeeting replay and start it at the time that person wrote that chat.
Import Whiteboard Data?

This allows you to set whether you wish to import the whioteboard data associated with the meeting. This is a list of whiteboard snapshots taken during the meeting.

If you do import this list there will be a 'Whiteboard' Map created containing a list of reference nodes listing at what time which snapshot was taken. When double-clicked each node will launch the Flashmeeting replay and the whiteboard at the time that the snapshot was taken.

Import Annotations Data?

This allows you to set whether you wish to import the annotations associated with the meeting. This is a list of annotations added to the meeting.

If you do import this list there will be an 'Annotations' Map created containing a list of reference nodes listing the time and the annotation that was added. When double-clicked each node will launch the Flashmeeting replay and start it at the time the annotation was added.
Import File Data?

This allows you to set whether you wish to import the file data associated with the meeting.

If you do import this list there will be a 'Meeting files' Map created containing a list of reference nodes listing the file names of the files associated with the meeting. When double-clicked each node will launch a browser pointing at the file to download.
Import Voting Data?

This allows you to set whether you wish to import the voting data associated with the meeting.

If you do import this list there will be an 'Voting' Map created containing a list of reference nodes listing the time of the vote, the person who votined and how they voted. When double-clicked each node will launch the Flashmeeting replay and start it at the time the vote was cast.
Mark nodes seen

Mark all imported nodes as already seen. This is part of the groupware functionality.

If you do not check this box all imported nodes will be marked as unseen and will have red borders around them until you have viewed thier contents or mared them as seen.

Once you have completed the import options, press the 'Import...' button. A file browser dialog window will open, (see figure 2 below), and you will need to locate the file that you wish to import. Only xml files will be accepted for importation.


Figure 2. The Import File Browser dialog


Once you have selected a file, and pressed 'Import', the importation will begin. You will see a progress bar showing you how the import is progressing.

N.B. All xml files must comply to the Flashmeeting dtd.
