Zoom dropdown menu

Access all the zoom options from the following drop-down menu.

 Zoom Out

Zoom out in stages.. 75%, 50% and 25%, depending where you start from.

 Zoom to 100%

Zoom to 100%

 Zoom In

Zoom in in stages..50% 75% 100%, depending where you start from.

 Zoom to Fit

Zoom so the whole map fit into the visible area.

If the map is extremely large this may not be practical if you window is set to be very small.

 Focus Node

Zoom to 100%, with the last focused node as the centre of the map.
Decrease the Node Label, list and side view text sizes by increments of one.
Set the Node Label, list and side view text sizes to thier normal size - respecting current zoom settings.
Increase the Node Label, list and side view text sizes by increments of one.
