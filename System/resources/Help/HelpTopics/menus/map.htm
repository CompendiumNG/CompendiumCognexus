Outline View (keyboard shortcut: Alt-V-O)

Opens the following submenu:

This allows you to open two outline views either just maps and lists, or all nodes. It opens as a tree panel on the left-hand side of the screen.

 
Unread View (keyboard shortcut:
Alt-V-U)		 Open a new view on the left-hand side of the screen which holds a tree menu listing all unread nodes and views. It is part of the Group Working functionality see Group Working - Unread View
Tag View (keyboard shortcut:
Alt-V-G)		 Open a new view on the left-hand side of the screen which holds a tree menu listing all tags. see Tag Management
Find a Map/List (keyboard shortcut:
Alt-V-M)		 Open a list of all the maps and lists in the current database  
Toolbars (keyboard shortcut: Alt+V+T Open a sub-menu where you can turn on and off the toolbars.  
Status Bar (keyboard shortcut: Alt+V+S) Hides and shows the Status Bar  
View History Bar (keyboard shortcut: Alt+V+B) Hides and shows the View History Bar, which displays a navigable path of the views opened to reach the current view.  
Aerial View (keyboard shortcut:
Alt-V-V)		 Open a small floating window displaying an aerial view. This window will always show an aerial view of the current map (it disappears if you focus a list, but will reappear again next time you focus a map).

If you close the aerial view window the facility is disabled, until you re-enable it on the Map menu.

The window is resizable and will scale the contents according to the window size you drag it to.

The contents are basically a scaled version of the parent map and therefore should have most the  functionality of the parent map.

You can add/edit/delete nodes and links etc and the two views should keep each other updated.

If you select a node in the Aerial view the parent view will scroll to show the node.

  

Zoom
(keyboard shortcut:
Alt-V-Z)

This opens a sub menu with zoom options that will zoom the currently selected view. Most options are obvious. When you choose the 'Focus Node' option, whatever zoom level you are at you will be re-zoom to 100% but with the last node that was selected central to the view.

This can be useful with big maps, where you can zoom out find a node then zoom in focused on that node.

Image Rollover
(keyboard shortcut:
Alt-V-I)

Switch on and off the image rollover facility. When this is activated, any nodes with external images are mouse sensitive. When you roll the move over them they will enlarge. If their full size is too large for the visible screen, they will be scaled to fit.

Auto Label Searching (keyboard shortcut: Alt-V-L) Switch on and off label completion, which means that as you type in a node label you will get a box showing possible matches.

There is keyboard navigation using the up/down arrow keys between the label and the list box.

Pressing the Enter key or double clicking on an item in the list transcludes the selected option, replacing the current node (which gets put in the trashbin).

F1 opens the node's contents dialog if you want more information.

  

Arrange
(keyboard shortcut:
Alt-V-R)

This opens a submenu with two options:

Left to Right: (Alt-V-R-R or Ctrl + R) Cleans up the nodes and links in the current view and arranges them in a neat tree pattern from left to right.

Top-Down: (Alt-V-R-W or Ctrl + W) Cleans up the nodes and links in the current view and arranges them in a neat tree pattern from the top of the view downward.

Note: You can undo and redo an Arrange.

 
Align   (keyboard shortcut: Alt+V+A) This option is only available when nodes are selected and opens a submenu with 6 node alignment options: left, center, right, top, middle and bottom.

This aligns the selected nodes according to the option chosen.

Note: You can undo and redo an Align.

  
Go To...  (keyboard shortcut: Alt+V+G)

The Go To menu option opens a dialog that enables users to enter a node ID and quickly go to that node. This feature is used with the web map email feedback feature enabled during web map exports.

When a user comments on a node in a web map, an email is generated containing the comment and the node ID. Copy and paste the node ID into the Go To dialog. The map containing the node opens and the node is highlighted, making it easy to copy and paste the user's comment where appropriate.
