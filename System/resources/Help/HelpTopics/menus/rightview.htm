The White-Space Right-Click Menu

The Map view right-click menu:



Menu option

For more information

Arrange

Access a submenu with arrange options.

Left to Right: Cleans up the nodes and links in the current view and arranges them in a neat tree pattern from left to right.

Top-Down:  Cleans up the nodes and links in the current view and arranges them in a neat tree pattern from the top of the view downward.

Note: You can undo and redo an Arrange.

 
Align This option is only available when nodes are selected and opens a submenu with 6 node alignment options: left, center, right, top, middle and bottom.

This aligns the selected nodes according to the option chosen.

Note: You can undo and redo an Align.

  

Create Node

Create a new node of a specific type in the current view

See Creating Nodes

Change Type To...

Change the type of the current node, or all selected nodes, to node type selected from menu.

See Nodes Types
Manage Stencils... Manage Stencil Sets. This opens a dialog which allows you to create / edit  and delete stencil sets and their items. See Stencils
Open Stencil Access a submenu holding a list of all currently available stencil sets. See Stencils
References... This option only appears if the view contains any reference nodes. It will lead to a submenu listing those references in order of reference type. When you select on it will open the reference with the appropriate external application.  

Copy

Copy a node, selection of nodes, link, selection of links, or text from a node Label or Detail.

When you Copy something, it is placed on your computer’s clipboard, available for subsequent actions. It is not removed from its current location.

Note: Subsequent Cut or Copy commands will replace whatever is on the clipboard.

 

Cut

Cut (remove) a node, selection of nodes, link, selection of links, or text from a node Label or Detail.

When you Cut something, it is removed from the current view or node and placed on your computer’s clipboard, available for subsequent actions. Cut nodes are placed in the Trash Bin, if they are not present in any other view.

Note: Subsequent Cut or Copy commands will replace whatever is on the clipboard.

 

Paste

Place objects (nodes, links, and/or text) from the clipboard into the current view or node.

 

Import

Export

Move Detail into Label

Move the detail text for all selected nodes into their label fields

 

Move Label into Detail

Move the label text for all selected nodes into the end of the first detail window.

 
Mark Seen

Mark the selected nodes as 'seen'. This is part of the group working functionality.

When other users add or edit nodes, you see them displayed with a red border to indicate that you should look at them as they have changed / are new. This option allows you to mark the selected nodes as seen. This will remove the red border from the nodes.

see Group Working
Mark Unseen

Mark the selected nodes as 'unseen'. This is part of the group working functionality.

see Group Working

See Node Contents and Properties

See Node Contents and Properties
