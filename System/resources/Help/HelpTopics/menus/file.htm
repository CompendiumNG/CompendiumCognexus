For more information

New
(keyboard shortcut:
Alt-F-N)

Create a new Compendium project.

See Creating a New Project

Open
(keyboard shortcut:
Alt-F-O)

Choose a Compendium project to work in.

See Opening a Project

Close
(keyboard shortcut:
Alt-F-C)

Close the current project.

 

System Settings
(keyboard shortcut:
Alt-F-S)

This opens a dialog which allows you to enter any relevant proxy setting for your PC. This is aprt of the new Group working functionality.

If you wish Compendium to use url's for node and background images and you are on a network that uses a proxy to connect to the Internet, you need to enter the proxy settings in the System Settings dialog. This will allow Compendium to access web images.

 see System Settings

Backup
(keyboard shortcut:
Alt-F-B)

Backup the currently open database.

See Backing a Project up

Database Administration
(keyboard shortcut:
Alt-F-A)

This opens a window which allows you to set up MySQL database connection profiles and switch Compendium to use one of them as it's data source or back to the default database supplied, Derby.

See Database Administration

Convert From MySQL To Derby
(keyboard shortcut:
Alt-F-M)

or

Convert From Derby To MySQL (keyboard shortcut: Alt-F-M)

You can use this facility to import a Compendium MySQL database project into Compendium Derby database project or a Compendium Derby Database project into a MySQL database project.

See Converting a MySQL Project

or

Converting a Derby Project

Project Management
(keyboard shortcut:
Alt-F-M)

This opens a window with several project managements facilities including backing up, restoring, renaming and deleting. You must be an administrator to access this options and it is only available when no projects are open.

See Project Management

Connections

Send To Jabber (keyboard shortcut: Alt-F-J or Command-F-J)

When active, this option will display a list of Jabber users to whom you can send the currently selected, node/nodes. This menu options will only be active once you have opened a Jabber connection.

See Jabber

Send To IX
(keyboard shortcut:
Alt-F-D)

When active, this option will display a list of IX Panel users to whom you can send the currently selected node/nodes. This menu option will only be active, once you have opened a IX Panel connection.

See IX Panels

Print
(keyboard shortcut:
Alt-F-P)

Send the currently selected view to your printer. Currently List views cannot be printed on the Mac platform.

 

Exit
(keyboard shortcut:
Alt-F-X)

Close the current database and exit the application.
