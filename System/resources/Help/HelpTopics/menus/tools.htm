The Tools Menu

(keyboard shortcut: Alt-T)



Menu option

Allows you to:

For more information:

User Manager
(keyboard shortcut:
Alt-T-U)

Administer users for the current database.

See User Administration.
Templates (keyboard shortcut:
Alt-T-P)		 This menu is dynamically created from the contents of your Compendium Templates folder. For each subfolder a submenu is created and any files are listed as menu items. You can have folders within folders. If you select a template file menu item, that template will be imported into the current open view.  
memetic (keyboard shortcut:Alt-T-M) This opens the memetic submenu holding two options:

'Manual Stop', to stop a memetic meeting recording/replay if the Meeting Manager fails to do so.

'Manual Upload', this opens the Meeting Upload dialog and allows you to upload meeting related files in the event that the automatic upload fails.

 See memetic
Launch UDig This open will try and launch your installation of the uDig GIS application, if you have one. See ecosensus
Manage Stencils... (keyboard shortcut:
Alt-T-S)		 Manage Stencil Sets. This opens a dialog which allows you to create / edit  and delete stencil sets and their items. See Stencils
Open Stencil (keyboard shortcut:
Alt-T-E)		 Access a submenu holding a list of all currently available stencil sets. See Stencils
Manage Link Groups (keyboard shortcut:
Alt-T-L)		 Manage Link Groups. This opens a dialog which allows you to create / edit  and delete link groups and their link types. See Link Groups
Restore Default Link Group (keyboard shortcut:
Alt-T-G)		 Restore the default IBIS link group. You can edit the default IBIS link group through the Link Group Management dialog. If at any time you wish to restore the original setting for this group, you can use this button to do so. See Link Groups
Scribble Pad      (keyboard shortcut:
Alt-T-A)

This opes a submenu with the following Scribble Pad options:

  • Activate Scribble Pad
  • De-activate Scribble Pad
  • Save Scribble Pad
  • Clear Scribble Pad
See Scribble Pad
Show Lost Nodes (keyboard shortcut: Alt-T-L) This options will open a dialog which will list any nodes that have somehow become detached from any view, but not marked for deletion.

This list should normally be empty. But from time-to-time, due to various factors including general bugs in Compendium, historically to do with cut and paste and deletion, odd nodes may appear here. From this dialog you can insert a given node into the current view and thereby release it from its 'limbo' state. From there you can always delete it if desired.

  

Reset Toolbars to Default
(keyboard shortcut:
Alt-T-R)

If you have moved your toolbars around to various side of the main screen, or reordered them, selecting this will reset them all to their default positions

See Toolbars
Project Options...
(keyboard shortcut:
Alt-T-J)

This opens the Project Options dialog (Project Preferences on the Mac).

Through this dialog you can set the node formatting defaults for newly created nodes.

See Project Options

User Options...
(keyboard shortcut:
Alt-T-O)

This opens the User Options dialog (User Preferences on the Mac).

Through this dialog you can set a vriety of options store on a per Compendium installation basis

See User Options
